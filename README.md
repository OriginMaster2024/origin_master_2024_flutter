# origin_master_2024_flutter

## Requirements

- [asdf](https://asdf-vm.com/)：バージョン管理ツール

  ```bash
  brew install asdf
  echo -e "\n. \"$(brew --prefix asdf)/libexec/asdf.sh\"" >> ~/.zshrc
  asdf plugin add flutter
  asdf plugin add ruby https://github.com/asdf-vm/asdf-ruby.git
  asdf install
  ```
